# Cloud Cost Budget Tracker

This project is a Python-based tool that calculates and analyzes monthly server costs against a user-defined budget.

## Features
- Accepts user input for number of servers and individual server costs
- Calculates total monthly cost
- Compares total cost with a given budget
- Displays whether the user is under, over, or exactly on budget

## Technologies Used
- Python

## Purpose
This project was built to practice programming concepts such as loops, functions, and conditional logic, while simulating a real-world cloud cost management scenario.

## How to Run

1. Download the file
2. Run using Python:

python cloud_cost_budget_tracker.py


## Example

How many servers do you have? 3
Enter monthly cost for Server 1: $50
Enter monthly cost for Server 2: $75
Enter monthly cost for Server 3: $100

Total cost: $225
You have $25 left in your budget.
